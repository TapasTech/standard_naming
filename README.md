# 标准命名

> 制定统一命名风格，减少前后端交互沟通

## 资讯文章类

|中文|英文|类型|
|:------|:----|:----|
|标题|title|string|
|简介|summary|string|
|描述|description|string|
|正文|content|string|
|作者|author|string|
|头像|avatar|string|
|发布日期|publish_at|string|
|发布状态|status|string|
|编辑|editor|string|
|标签|tags|array(string)|
|图片、头图|image|string|
|来源|origin|string|
|来源链接|origin_url|string|
